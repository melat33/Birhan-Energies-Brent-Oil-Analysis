 📊 Overview
A comprehensive full-stack dashboard for analyzing Brent Crude Oil prices with interactive visualization of geopolitical events, statistical change points, and market impacts from 1987-2022.

Task 3: Interactive Dashboard Development** - This project demonstrates the integration of a Flask backend API with a React frontend dashboard for data visualization and analysis.

✨ Key Features

 🎨 Frontend Dashboard
- **Interactive Price Charts** with event highlighting
- **Date Range Filters** with quick-select options
- **Event Timeline** with click-to-zoom functionality
- **Responsive Design** that works on all devices
- **Real-time Data Updates** from the backend API
- **Export Capabilities** for price data
- **Comprehensive Metrics Display**

### ⚙️ Backend API
- **RESTful API** with comprehensive endpoints
- **Data Filtering** by date range and categories
- **Statistical Analysis** endpoints
- **Health Monitoring** and status reporting
- **CORS Enabled** for frontend integration
